ajaxtestのなかでnpmを実行しています
トップ画面のURLはhttp://localhost:3000/になっています

トップ画面にリストが表示されます。
リスト名、達成率、○個中○個チェック済みか、作成日、一番近い期限日、デリートボタンが表示されています。
リストでは、○個中○個チェック済みかをパーセントに変換し、表示しています。
トップ画面を開いて、すぐにTodoの進行状況がわかるように、その数値を可視化し、棒グラフのようにしています。
デリートボタンを押すと、リストごと消すことができます。
リストをクリックすると、Todo画面に移動します。

Todo画面では、Todoを追加するとき2つの種類があります。Have to do（以下「Have」とする）とWantto do（以下「Want」とする）の2種類があり、"todo切り替え"のボタンで切り替えられます。
Haveは、やらなければいけないこと、Wantは、余力があればやりたいこと。Wantのほうは、やらなくてもいいため、トップ画面の○個中○個のところには、カウントされません。

Todoは、
todo名、作成日、期限日、完了か末完了か、デリートボタンが表示されます。
クリックすることで、完了か見完了か切り替えられます。
todoによって背景色が、変わっています。
Haveの完了が緑色、Haveの末完了が赤色、Wantの場合、完了でも、未完了でも、青色で表示されます。

デリートボタンを押すと、そのTodoを消すことが出来ます。

トップ画面の○個中○個のところや、期限のところは、
Todoを追加したときや、Todoをクリックした時に、更新されます。


検索画面では、
キーワード検索と日付検索があります。
上のフォームにテキストを入力し”キーワードでさがす”のボタンを押すと検索できます。
検索結果は、Todo名に入力した文字を含むものが、作成日が新しい順に表示されます。

下のフォームでは、いつからいつまでを指定でき、”日付でさがす”ボタンを押すと検索できます。
検索結果は、期限日が指定した範囲内のものが表示され、
期限の日付が近い順に表示されます。


Todoは、
Todo名、どこのリストか、作成日、期限が表示されます
背景色は、Todo画面と同じパターンで色がついています。
